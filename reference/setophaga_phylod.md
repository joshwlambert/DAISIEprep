# A phylogenetic tree of setophaga species with endemicity status as tip states.

A phylogenetic tree of setophaga species with endemicity status as tip
states.

## Usage

``` r
setophaga_phylod
```

## Format

### `setophaga_phylod`

A `phylo4d` object (from the phylobase package) with 19 tips and 18
internal nodes.

## Source

Valente, L.M., A.B. Phillimore and R.S. Etienne (2015). Equilibrium and
non-equilibrium dynamics simultaneously operate in the Galapagos
islands. Ecology Letters 18: 844-852.

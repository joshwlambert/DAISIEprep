# Checks the validity of the Multi_island_tbl class

Checks the validity of the Multi_island_tbl class

## Usage

``` r
check_multi_island_tbl(object)
```

## Arguments

- object:

  Instance of the Multi_island_tbl class

## Value

Boolean or errors

## Examples

``` r
multi_island_tbl <- multi_island_tbl()
check_multi_island_tbl(multi_island_tbl)
#> [1] TRUE
```

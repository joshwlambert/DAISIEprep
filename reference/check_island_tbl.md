# Checks the validity of the Island_tbl class

Checks the validity of the Island_tbl class

## Usage

``` r
check_island_tbl(object)
```

## Arguments

- object:

  Instance of the Island_tbl class

## Value

Boolean or errors

## Examples

``` r
island_tbl <- island_tbl()
check_island_tbl(island_tbl)
#> [1] TRUE
```

# Checks the validity of the Island_colonist class

Checks the validity of the Island_colonist class

## Usage

``` r
check_island_colonist(object)
```

## Arguments

- object:

  Instance of the island_colonist class

## Value

Boolean or errors

## Examples

``` r
island_colonist <- island_colonist()
check_island_colonist(island_colonist)
#> [1] TRUE
```

# DAISIEprep: Extracts Phylogenetic Island Community Data from Phylogenetic Trees

Extracts colonisation and branching times of island species to be used
for analysis in the R package 'DAISIE'. It uses phylogenetic and
endemicity data to extract the separate island colonists and store them.
